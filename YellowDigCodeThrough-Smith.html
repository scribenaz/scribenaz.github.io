Introduction

This Code Through will introduce you to a package called ‘datapasta’. This package will allow you to take data from a website and copy/paste it into R as a tibble or dataframe instead of building a dataset from scratch and saving you time. This package will also create tibbles and can be used to copy data from Excel files as well as Jupyter files.

Content Overview

I will be using data from the Las Vegas Review-Journal on the Las Vegas Raiders football statistics. The website publishes various statistics throughout the year on the Las Vegas Raiders. This demonstration will focus on the copy/paste function of this package from a live website.

Package needed: datapasta
